Usability Scale

Learnability: The website is simple, easy to use and navigate, everything is clearly defined and won't be likely to cause any confusion.

Efficiency: Everything in the website gets right to the point and so performing task will be swift ans easy.

Memorability: The website is so cleverly designed to be unique and simple that gets down to the point that the user will not even need to memorize what to do and where.

Errors: Users will run into error when filling up the form for the newsletter when they put in the wrong input but the website will let the user know where the mistake was made.

Satisfaction: The website is very welcoming; the designs and colors used for the website are perfect for a bakery.
